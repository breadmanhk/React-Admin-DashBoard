● 这是一个React 管理员仪表板项目，具有以下特点：

  技术栈

  - React + Vite - 现代前端框架和构建工具
  - Tailwind CSS - 实用优先的 CSS 框架
  - Recharts - 数据可视化图表库
  - Framer Motion - 动画库
  - React Router - 路由管理
  - Lucide React - 图标库

  项目结构

  包含 7 个主要页面：
  - Overview - 总览页面
  - Products - 产品管理
  - Users - 用户管理
  - Sales - 销售数据
  - Orders - 订单管理
  - Analytics - 数据分析
  - Settings - 系统设置

  功能组件

  - 响应式侧边栏导航
  - 各种数据可视化图表（销售趋势、用户增长、订单分布等）
  - 用户活动热力图
  - AI 洞察分析
  - 设置页面（个人资料、通知、安全等）

  这是一个完整的现代化管理后台界面套件，适用于各种 Web 应用的管理面板需求。
